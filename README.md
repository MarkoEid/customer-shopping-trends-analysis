# 🛍️ Customer Shopping Trends & Sales Analytics

## 📖 Project Overview
This project provides a comprehensive look at consumer behavior within the **Retail and E-commerce domain**. By combining **Exploratory Data Analysis (EDA)** in Python with a professional **Sales Dashboard**, this repository demonstrates the ability to transform raw transactional data into actionable business intelligence.

---

## 📂 Dataset Information
* **Source:** [Kaggle - Customer Shopping Trends Dataset](https://www.kaggle.com/datasets/iamsouravbanerjee/customer-shopping-trends-dataset)
* **Domain:** Retail & E-commerce
* **Data Characteristics:** The dataset consists of 3,900 records capturing various customer attributes and purchasing habits.

### 📋 Column Glossary
| Column | Description |
| :--- | :--- |
| **Customer ID** | Unique identifier for each customer |
| **Age / Gender** | Demographic information of the shopper |
| **Item Purchased** | Specific product name |
| **Category** | Product group (e.g., Clothing, Footwear, Electronics) |
| **Purchase Amount** | Total value of the transaction (USD) |
| **Location** | State/Region where the purchase was made |
| **Subscription Status** | Indicates if the user is a premium member (Yes/No) |
| **Payment Method** | Preferred method (Credit Card, PayPal, Cash, etc.) |
| **Shipping Type** | Choice of shipping (Standard, Express, Next Day) |

## 📊 Phase 1: Interactive Sales Dashboard
The dashboard summarizes macro-level performance across the Indian market.

### 🚀 Key Performance Indicators (KPIs)
* **Total Revenue:** 3M  
* **Total Units Sold:** 34K  
* **Regional Leaders:** Maharashtra (542K) & Delhi (492K) lead in sales volume.

### 🔍 Visual Insights
* **Regional Reach:** The **North (35.1%)** and **West (31.2%)** regions are the primary revenue drivers.
* **Product Performance:** **Nestle Smarties Pop-Up** (15.8%) and **Milky Bar Stick** (13.1%) are the top contributors.
* **Holiday Spike:** Data shows a significant exponential rise in sales during **November and December**.

## 🧪 Phase 2: Python Data Analysis (EDA)
The Jupyter Notebook (`7-th assignment(Shopping_trends).ipynb`) explores the underlying patterns of the Kaggle dataset.

### 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** `Pandas` (Cleaning), `Seaborn` & `Matplotlib` (Visualization).

### 💡 Core Findings
* **Demographic Split:** Balanced gender distribution with high engagement across middle-age groups.
* **Payment Preferences:** Identified a strong preference for digital payments (Credit Card/PayPal) among high-value transactions.
* **Subscription Impact:** Analyzed how subscription status influences "Average Purchase Amount" and "Frequency of Purchases."
